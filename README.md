# Slither Clone

Klon gry [slither.io](https://slither.io) napisany w HTML, CSS i JavaScript.

## Uruchomienie

Otwórz plik `index.html` w przeglądarce — nie wymaga serwera ani instalacji zależności.

## Sterowanie

| Akcja | Klawisz / mysz |
|---|---|
| Kierunek | Ruch myszy |
| Boost | LPM, Spacja lub Shift |
| Mobilne | Dotyk = kierunek, trzymanie = boost |

## Funkcje

- Sterowanie wężem i mechanika boostu
- Boty z AI (polowanie, ucieczka, zbieranie jedzenia)
- Kolizje tylko z innymi wężami
- Okrągła arena z widoczną granicą mapy
- Ranking Top 10 i minimapa

## Struktura

```
slither-clone/
├── index.html
├── css/style.css
└── js/game.js
```
